PLACEHOLDER ICONS
==================

The icon.svg file is a template. You need to convert it to PNG format in the following sizes:

- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

You can use online tools like:
- https://www.iloveimg.com/resize-image
- https://www.adobe.com/express/feature/image/resize
- Or use ImageMagick: convert icon.svg -resize 16x16 icon16.png

For now, the extension will not load properly without these PNG files.
You can also design custom icons using tools like Figma or Canva.
