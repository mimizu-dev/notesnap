Connect to Obsidian

This extension saves notes into your Obsidian vault using the Local REST API community plugin.

  1. 1
    Open Obsidian → Settings → Community plugins
    Turn off Restricted mode, click Browse
  2. 2
    Search Local REST API → Install → Enable
  3. 3
    Go to Settings → Local REST API to find the URL and API Key:
    HTTP http://127.0.0.1:27123

    Enable it in Obsidian → Settings → Local REST API → "Enable Non-encrypted (HTTP) Server".

    Copy either URL and the API Key, then click next.
⚠️ Keep Obsidian open whenever you sync. To use a different vault, just switch vaults in Obsidian.

Default port is 27123 — shown in Settings → Local REST API.

The name of your Obsidian vault — shown in its title bar or when you switch vaults.
